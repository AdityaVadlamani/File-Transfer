File-Tranfer package
==================

This project allows a user to simulate a file transfer service with multiple terminals on their local device.

Installation
------------

Use the package manager `pip <https://pip.pypa.io/en/stable/>`__ to
install bidict.

.. code:: bash

    pip install bidict
